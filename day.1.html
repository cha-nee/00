  1. < strong > < /strong >
    2. ➡️ 글씨체를 볼드체로 변경
  2. < u > < /u >
    3. ➡️ u = underline 의 약자 (글자 밑줄)
  3. < h1~h6 > < /h1~h6 >
    4. ➡️ head 의 크키 1이 가장 큰 글자이고, 6으로 갈수록 작아진다.
  4. < br >
    5. ➡️ 단순 줄 바꿈으로 키보드 enter과 같은 역할, 줄과 줄 사이의 여백이 없다.
  5. < p > < /p >
    6. ➡️ p = paragraph 의 약자, 단락을 나누어 준다.
  6. < img >
    7. ➡️ 이미지를 불러옴, 혼자서는 아무 역할을 하지 않아서 불러올 source가 필요하다.
    < img src="파일명" > 의 형태
  7. < li > < /li > / < ul > < /ul >,< ol > < /ol >
    8. ➡️ 자식과 부모태그로 항상 함께 붙어있다.
    ➡️ li = list : 목차 나열
    ➡️ ul = unordered list : 그냥 정열
    ➡️ ol = ordered list : 숫자로 정열
